## Usage
```
usage: hko_report.py [-h] [-l] [-9] [-a] [--all] [--english]

Print weather report from Hong Kong Observatory.

optional arguments:
  -h, --help       show this help message and exit
  -l, --latest     show latest general weather information
  -9, --nine-days  show 9 days weather forecast
  -a, --astron     show astronomical observation
  --all            show all information
  --english        show information in English
```
